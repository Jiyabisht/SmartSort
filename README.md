# SmartSort
A real-time smart search engine that compares product prices across Amazon and Flipkart to help users find the best deals instantly.

# Features
🔍 Real-Time Price Comparison
Fetches and compares product prices from Amazon and Flipkart in real time.

🧹 Duplicate Elimination
Removes repeated or irrelevant product listings for cleaner search results.

💰 Best Deal Detection
Identifies the most desirable deals based on pricing and product similarity.

🧠 Memory Cache System
Stores recent searches for up to 10 minutes to improve response time and reduce repeated requests.

🤖 Fuzzy Matching Algorithm
Detects similar products even when names are slightly different.
